## UI Direction Approval
- Chosen visual direction: **Modern Family-inspired warm palette + 3D art-heavy interface**
- Motion intensity: **High (always-on layered parallax + depth transitions)**
- Target behavior:
  - Home starts on Today with clear status lanes.
  - 3D interactions are visible by default (cards, hero element, scene layers).
  - Web/mobile parity with safe fallbacks where platform APIs differ.
